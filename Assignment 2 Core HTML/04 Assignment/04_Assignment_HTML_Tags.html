List of HTML tags:

  1. h1-h6:Defines heading of different levels.where h1 is the most important and h6 is the lowest heading.
  2. p:Represents a paragraph of text.
  3. a: Creates hyperlinks, allowing you to link to other web pages or resources.
  4. img:Embeds images into the HTML document.
  5. br:Represents a line break, forcing content to appear on the next line.
  6. hr:Represents a horizontal rule or line, used to separate content.
  7. div:Represents a division or section in an HTML document.
  8. span:Represents an inline container used to apply styles or scripting to a specific section of text.
  9. table: Defines an HTML table
