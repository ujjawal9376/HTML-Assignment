HTML 5 Features Showcase

Feature 1: Video Playback

HTML5 video can use CSS and CSS3 to style the video tag. You can change the border, opacity, reflections, gradients, transitions, transformations, and even animations. HTML5 makes adding video super-fast and without having to build a video player.

Feature 2: Nav Tag

The <nav> tag defines a set of navigation links. It is used for the part of an internet site that links to different pages at the website. The hyperlinks can be organized through a number of approaches.

Feature 3: Progress Tag

The progress tag is used to check the progress of a task during the execution. Progress tag can be used with the conjunction of JavaScript.
